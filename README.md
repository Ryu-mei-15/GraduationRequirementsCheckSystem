# 🎓 卒業要件チェックシステム

## 概要

これは，情報学科の学生が卒業要件を満たしているかを確認するための Streamlit アプリケーションです．
履修済みの科目にチェックを入れることで，卒業に必要な単位数や各区分の要件をどれだけ満たしているかをリアルタイムで確認できます．

## ✨ 主な機能

*   **履修状況の可視化**: 共通教育科目，専門教育科目などのカテゴリごとに取得単位数を表示します．
*   **卒業要件の自動判定**: 卒業に必要な総単位数，各科目の必修要件，専門トラックの要件などを自動で判定します．
*   **進捗の保存・読み込み**: チェックした履修状況をファイルに保存し，後から読み込んで再開することができます．
*   **リアルタイムフィードバック**: 科目をチェックするたびに，サイドバーの判定結果が即座に更新されます．

## 🚀 使い方

### 1. 準備

このアプリケーションは Python と Streamlit ライブラリを使用します．
まず，ターミナル（またはコマンドプロンプト）で以下のコマンドを実行し，必要なライブラリをインストールしてください．

```bash
pip install streamlit
```

### 2. アプリケーションの実行

`app.py` ファイルがあるディレクトリで，ターミナルを開きます．
そして，以下のコマンドを実行してください．

```bash
streamlit run app.py
```

コマンドを実行すると，Webブラウザが自動的に起動し，アプリケーションが表示されます．

## 画面説明

### メイン画面

*   **① 共通教育科目**: 語学，人文，社会，自然などの共通教育科目をチェックします．
*   **② 専門基礎・PBL**: プログラミング基礎やPBL（Project Based Learning）科目など，専門分野の基礎となる科目をチェックします．
*   **③ 専門トラック**: 「データサイエンス」「ICT」「人間・社会情報」の3つの専門トラックから，履修状況に応じて科目をチェックします．
*   **④ その他・他学部**: 単位互換科目や他学部の科目など，その他の科目をチェックします．

### サイドバー

*   **データ保存**: 「現状を保存する」ボタンをクリックすると，現在のチェック状況が `graduation_data.json` というファイルに保存されます．
*   **判定結果**: 取得単位数の合計や，各要件の達成状況がリアルタイムで表示されます．卒業要件を満たしていない場合は，未修得の必修科目などが表示されます．

## 📄 データファイルについて

履修状況は `graduation_data.json` という名前のJSONファイルに保存されます．このファイルを削除すると，保存したデータが消えてしまうのでご注意ください．
